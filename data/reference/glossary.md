# Glossary

Acronyms and terms used across the space-ops corpus. Synthetic hackathon data.

| Term | Meaning |
| --- | --- |
| **CDRA** | Carbon Dioxide Removal Assembly. The ECLSS subsystem that scrubs CO2 from cabin air using cycling sorbent beds. |
| **DCM** | Display and Control Module. The chest-mounted control panel on the EMU spacesuit. |
| **ECLSS** | Environmental Control and Life Support System. Cabin air, water, and atmosphere management. |
| **EMU** | Extravehicular Mobility Unit. The spacesuit worn during a spacewalk. |
| **ETCS** | External Active Thermal Control System. Ammonia coolant loops that reject heat externally. |
| **EVA** | Extravehicular Activity. A spacewalk outside the pressurized vehicle. |
| **Egress** | Exiting the airlock/vehicle to begin an EVA. |
| **Ingress** | Returning into the airlock/vehicle. |
| **LOS** | Loss Of Signal. Period with no communication link to ground control. |
| **ORU** | Orbital Replacement Unit. A modular component designed to be swapped in space. |
| **OGA** | Oxygen Generation Assembly. Produces breathing oxygen aboard the station. |
| **PGT** | Pistol Grip Tool. The cordless, torque-settable power tool used for EVA fasteners. |
| **Pre-breathe** | Breathing pure O2 before an EVA to purge nitrogen and prevent decompression sickness. |
| **psia** | Pounds per square inch, absolute. Pressure unit used throughout. |
| **QD** | Quick Disconnect. A mate/demate fluid or coolant line coupling. |
| **R&R** | Remove and Replace. |
| **SAFER** | Simplified Aid For EVA Rescue. A small self-rescue jetpack worn during EVA. |
| **Safing** | Placing a system in a known safe, de-energized/vented state before hands-on work. |
| **Sorbent bed** | The material bed in the CDRA that adsorbs CO2, then releases it when heated (desorb). |
| **Adsorb / Desorb** | CDRA bed cycle phases: capturing CO2 (adsorb) vs. baking it off (desorb). |
| **Tether** | Safety line clipping the crewmember to the structure during EVA. |
